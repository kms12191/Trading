- KR 모델의 DART 공시 피처 효과 검증

## 결론

1. KR은 DART 포함 모델을 유지한다.
   - DART 제거 대비 제약형 정책에서 초과수익 0.0101 대 0.0048, Precision 0.5217 대 0.4390, MDD -0.1281 대 -0.1948로 우위다.

2. KR 현재 Composite 정책은 너무 보수적이다.
   - 기준 정책에서 선택 행이 0개라 실제 추천 후보를 만들지 못한다.
   - 제약형 정책 후보는 후보 생성 문제를 완화하지만, 같은 검증 구간에서 탐색된 값이므로 바로 서빙 반영하면 안 된다.

3. US는 정책 개선 여지가 크다.
   - 현재 Composite는 초과수익 -0.0278, MDD -0.5447이다.
   - 제약형 후보는 초과수익 0.0138, Precision 0.6042로 개선됐다.
   - 단, US 원천 데이터가 최신성이 낮은 상태였으므로 fresh data 재수집 후 재검증이 필요하다.

## 권장 후속 작업

1. 제약형 정책 탐색을 재사용 가능한 스크립트로 승격한다.
   - 현재 결과는 일회성 실험 산출물이다.
   - `ml/src/tune_stock_policy_constrained.py` 형태로 고정 조건, seed, 출력 포맷을 코드화해야 한다.

2. Walk-forward 검증을 추가한다.
   - 정책 탐색 구간과 최종 평가 구간을 분리해야 한다.
   - 이 검증 전에는 서빙 정책으로 승격하지 않는다.

3. US 데이터를 최신화한 뒤 재실험한다.
   - US 기준 성능은 데이터 최신성 문제를 포함한다.
   - 최신 Toss US 캔들 수집 후 같은 실험을 다시 실행해야 한다.

4. 통과 기준을 모델 레지스트리 승격 조건에 연결한다.
   - 최소 선택 행, 최소 테스트 기간, 최소 초과수익, 최대 허용 MDD를 정책 승격 조건으로 명시한다.
