# AI 위탁운용 코인 숏 모델 구현 계획

상태: 2026-07-22 진행 중

## 목표

하락 수익 예측용 독립 모델을 자동 학습하고, 펀딩비를 포함한 성능검증 결과를 기존 위탁운용 대시보드에 표시한다.

## 작업

- [x] 숏 모델 설정과 전용 산출물 경로를 추가한다.
- [x] 숏 방향 승률 및 펀딩비 차감이 정확한지 백테스트 단위 테스트를 추가한다.
- [x] 파이프라인 번들, 수동 학습 API, 스케줄러 프리셋에 숏 모델을 연결한다.
- [x] 성능 요약 API와 기존 대시보드 읽기 전용 상태 영역을 추가한다.
- [x] 실제 학습과 백테스트를 실행하고 결과를 기록한다.

## 1차 성능 결과

- 검증 행: 115,410건, 숏 선택: 1,399건
- ROC-AUC: 0.608, 시간순 교차검증 평균 AUC: 0.588
- 비용 전 숏 방향 적중률: 60.7%, 비용 후 순승률: 50.2%
- 비용 반영 평균 수익률: -0.10%, 최대 낙폭: -99.8%
- 결론: 수수료·유동성 기반 슬리피지·펀딩비를 포함하면 기준을 충족하지 못하므로 `LIVE_TRADING_HOLD`다. 실거래는 연결하지 않는다.

## 안전 조건

이번 구현은 연구·모의검증만 포함한다. 바이낸스 선물 주문, 레버리지, 증거금 관리는 연결하지 않는다.
