# AI 위탁운용 코인 숏 모델 설계

상태: 2026-07-22 구현 진행 중

## 목적

기존 하락 위험 필터와 분리된 코인 선물 숏 전용 예측 모델을 자동 학습한다. 이 모델은 하락 수익 가능성을 검증하기 위한 연구·모의검증 구성요소이며, 이번 단계에서 바이낸스 선물 실주문을 실행하지 않는다.

## 모델과 검증

- 모델 버전: `lgbm_crypto_short_v1`
- 학습 목표: `residual_risk_label`
- 입력: 기존 30분 코인 피처와 펀딩비, 김치프리미엄, 시장 뉴스 피처
- 진입: 예측 확률이 `short_entry_threshold` 이상인 종목만 `SHORT`로 평가한다. 초기 기준은 최근 검증 구간의 확률 범위를 확인해 `0.35`로 둔다.
- 비용: 거래 수수료, 유동성 기반 슬리피지, 보유 구간 펀딩비를 숏 순수익에서 차감한다.
- 지표: 순평균 수익률, 순승률, 최대 낙폭, 선택 건수, ROC-AUC, 시간순 교차검증 AUC를 저장한다.

## 자동 학습 흐름

`crypto-v10-full` 프리셋은 상승 모델, 위험 모델, 숏 모델을 순서대로 학습·평가한다. 마지막으로 기존 상승 백테스트와 별도로 숏 전용 백테스트를 실행해 `crypto_backtest_short_v1.json`에 결과를 남긴다.

## 대시보드

기존 `AdminAiFundDashboard`에 읽기 전용 "바이낸스 선물 숏 모델" 상태 영역을 추가한다.

- 학습 파일이 없으면 "학습 대기"를 표시한다.
- 백테스트 결과가 없으면 "검증 데이터 없음"을 표시한다.
- 순수익, 순승률, 최대 낙폭, 선택 건수와 비용 가정을 한국어로 표시한다.
- 성능이 기준을 통과하지 않으면 "실거래 보류"를 표시한다.
- 이 화면은 바이낸스 선물 주문 기능을 켜지 않으며, 기존 현물 자동선별·주문 흐름을 바꾸지 않는다.

## 범위 밖

- 수익 보장 또는 투자 추천
